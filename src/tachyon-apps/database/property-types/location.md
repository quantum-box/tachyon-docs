# 位置情報型 (Location)

位置情報型は、地理的な位置データを格納するためのプロパティタイプです。緯度と経度の組み合わせによって地球上の特定の地点を表現します。

## データ形式

位置情報型は以下のJSONオブジェクト形式で格納されます：

```json
{
  "latitude": 35.6812362,
  "longitude": 139.7649361
}
```

- `latitude`: 緯度（-90.0～90.0の範囲の数値）
- `longitude`: 経度（-180.0～180.0の範囲の数値）

## 使用例

位置情報型は以下のようなユースケースで活用できます：

- 店舗・施設の位置情報管理
- イベント開催地の登録
- 配送先・配送元の位置情報記録
- 位置ベースの検索機能の実装

## 制約事項

- 緯度は-90.0～90.0の範囲に制限されます
- 経度は-180.0～180.0の範囲に制限されます
- 値は常に数値として扱われます（文字列ではありません）
