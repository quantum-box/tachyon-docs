# Tachyon Apps

## Business Logic Packages

### コアビジネス機能

- `llms`: LLMサービスの運用管理、評価、ベンチマーク、A/Bテスト機能
- `crm`: 顧客管理、顧客とのやり取り履歴、顧客セグメンテーション機能
- `projects`: プロジェクト管理、タスク割り当て、進捗管理機能
- `taskflow`: ワークフロー定義、タスク自動化、プロセス管理機能
- `order`: 注文処理、注文状態管理、注文履歴管理機能
- `payment`: 決済処理、支払い管理、請求書生成機能
- `delivery`: 配送管理、配送状況追跡、配送最適化機能
- `catalog`: 商品カタログ管理、商品情報管理、在庫管理機能
- `database`: スキーマレスデータベース作成・管理機能、動的データモデリング

### ユーザー管理・認証

- `auth`: ユーザー認証、認可、権限管理、セッション管理機能、APIキー発行、サービスアカウント作成
- `onboarding`: ユーザーオンボーディング、初期設定支援機能
- `feature_flag`: 機能の段階的リリース、ユーザーごとの機能制御機能

### データ管理・分析

- `corporate_db`: 企業情報管理、取引先データベース管理機能
- `csv_importer`: データインポート、一括データ処理機能
- `document_updater`: ドキュメント管理、バージョン管理機能
- `source_explore`: コード分析、依存関係管理機能
- `analytics`: プラットフォーム売上・利益集計、KPI ダッシュボード機能

### 通知・コミュニケーション

- `notification`: ユーザー通知、ビジネスイベント通知、アラート管理機能
- `providers`: 外部サービス連携、API統合管理機能

### プロセス管理

- `process_manager`: ビジネスプロセス管理、ワークフロー制御機能

### メインアプリケーション

- `tachyon_apps`: 全ビジネス機能を統合したメインアプリケーション
