# 調達・価格設定システムUIコンポーネント仕様

## 調達管理コンポーネント

### ProcurementPriceList
**場所**: `/procurement/components/ProcurementPriceList.tsx`

**機能**:
- 調達原価の一覧表示
- フィルタリング（リソースタイプ、サプライヤー）
- 統計情報表示（総数、サプライヤー数、平均価格）
- 階層価格の可視化

**主要な要素**:
```tsx
// 統計カード
- 総アイテム数
- サプライヤー数
- 平均価格（通貨別）

// フィルターセクション
- リソースタイプ選択（ドロップダウン）
- サプライヤー選択（ドロップダウン）

// 価格テーブル
- カラム: リソース、サプライヤー、基本価格、階層価格、有効期間、説明
- 階層価格の展開表示
- 通貨記号の適切な表示（$, ¥）
```

### SupplierList
**場所**: `/procurement/suppliers/components/SupplierList.tsx`

**機能**:
- サプライヤー情報の管理
- ステータス別フィルタリング
- 検索機能
- 連絡先情報の表示

**主要な要素**:
```tsx
// 統計カード
- サプライヤー総数
- アクティブ数
- 非アクティブ数

// 検索・フィルター
- 名前検索（テキスト入力）
- ステータスフィルター（Active/Inactive）

// サプライヤーテーブル
- カラム: 名前、連絡先、住所、ステータス、登録日
- 連絡先はメール・電話のリンク付き
- ステータスバッジ表示
```

### SupplyContractList
**場所**: `/procurement/contracts/components/SupplyContractList.tsx`

**機能**:
- 調達契約の一覧管理
- ステータス別統計表示
- サプライヤー別フィルタリング

**主要な要素**:
```tsx
// 統計カード（4列）
- 契約総数
- アクティブ
- ドラフト
- 期限切れ

// フィルター
- ステータス選択（ACTIVE/DRAFT/EXPIRED）
- サプライヤー選択

// 契約テーブル
- カラム: 契約番号、契約名、サプライヤー、契約期間、ステータス、説明
- ステータスバッジ（アイコン付き）
- 契約期間の分かりやすい表示
```

## 価格設定コンポーネント

### PricingPolicyList
**場所**: `/pricing/components/PricingPolicyList.tsx`

**機能**:
- 価格ポリシーのカードグリッド表示
- ポリシー作成・編集
- ステータス管理

**主要な要素**:
```tsx
// ポリシーカード
- ポリシー名（大きく表示）
- マークアップ率（基本/最小/最大）
- ステータスバッジ
- ルール数の表示
- 編集・詳細ボタン

// 新規作成ボタン
- 右上に配置
- PlusCircleアイコン付き
```

### PricingPolicyDialog
**場所**: `/pricing/components/PricingPolicyDialog.tsx`

**機能**:
- 価格ポリシーの作成・編集フォーム
- バリデーション
- プレビュー機能

**フォーム要素**:
```tsx
- ポリシー名（必須）
- 説明（テキストエリア）
- 基本マークアップ率（スライダー + 数値入力）
- 最小マークアップ率
- 最大マークアップ率
- マークアップ率のビジュアル表示
```

### PricingRuleList
**場所**: `/pricing/[policyId]/components/PricingRuleList.tsx`

**機能**:
- ルールの優先度順表示
- ルールタイプ別アイコン
- 有効期間の表示

**テーブル構成**:
```tsx
- 優先度（ドラッグ可能）
- ルールタイプ（アイコン + ラベル）
- 条件（JSONの要約表示）
- 調整値（％または固定値）
- 有効期間
- アクション（編集・削除）
```

### PricingRuleDialog
**場所**: `/pricing/[policyId]/components/PricingRuleDialog.tsx`

**機能**:
- ルール作成・編集
- 条件設定（JSONエディタ）
- プレビュー機能

**フォーム要素**:
```tsx
// 基本情報
- ルールタイプ選択
- 優先度（数値）
- 有効期間設定

// 条件設定
- JSONエディタ（Monaco Editor使用）
- テンプレート選択
- バリデーション表示

// 価格調整
- 調整タイプ（パーセント/固定/上書き）
- 調整値入力
```

### PriceSimulator
**場所**: `/pricing/[policyId]/components/PriceSimulator.tsx`

**機能**:
- リアルタイム価格計算
- ルール適用の可視化
- 計算過程の表示

**UI構成**:
```tsx
// 入力セクション
- リソースタイプ選択
- 数量入力
- 顧客セグメント選択

// 計算結果
- 基本価格
- 適用されたルール（順番に表示）
- 各ルールの影響額
- 最終価格（大きく表示）

// ビジュアル表示
- 価格変化のウォーターフォールチャート
```

### CustomerSegmentList
**場所**: `/pricing/segments/components/CustomerSegmentList.tsx`

**機能**:
- 顧客セグメントの管理
- 割引率の設定
- セグメント条件の定義

**カード表示**:
```tsx
- セグメント名
- 説明
- 割引率（％表示）
- 適用条件の要約
- 編集・削除ボタン
```

### PriceAnalysis
**場所**: `/pricing/analysis/components/PriceAnalysis.tsx`

**機能**:
- 価格分析ダッシュボード
- 収益トレンド表示
- 最適化提案

**ダッシュボード要素**:
```tsx
// KPIカード
- 総収益
- 平均マークアップ率
- アクティブポリシー数

// チャート
- 収益トレンド（LineChart）
- マークアップ率分布（BarChart）
- リソース別収益（PieChart）

// 最適化提案
- 提案リスト（優先度順）
- 期待効果の表示
- 実行ボタン
```

## 共通UIパターン

### ステータスバッジ
```tsx
// 色とアイコンの組み合わせ
Active: 緑色 + CheckCircleアイコン
Draft: 黄色 + Clockアイコン
Expired: 赤色 + XCircleアイコン
Archived: グレー + Archiveアイコン
```

### 空の状態表示
```tsx
// 共通パターン
- 中央配置のメッセージ
- 説明テキスト
- アクションボタン（新規作成など）
- イラストレーション（オプション）
```

### 価格表示
```tsx
// フォーマット
- 通貨記号を前に表示（$10.00, ¥1,000）
- 小数点以下2桁まで表示
- 3桁ごとにカンマ区切り
- 単位を併記（/百万トークン）
```

### フィルターUI
```tsx
// 共通構成
- Select/ComboBoxコンポーネント使用
- "すべて"オプションを最初に配置
- 選択時に即座にフィルタリング
- クリアボタン付き
```

### データテーブル
```tsx
// 共通機能
- ソート可能なヘッダー
- ページネーション（未実装）
- 行選択（チェックボックス）
- アクション列（右端）
- レスポンシブ対応
```
