# AIChatサービス概要

AIChatはTachyon Appsの一部として提供されるチャットボットサービスです。AIアシスタントとの対話を通じて、ユーザーが様々な質問や作業を行えるようにするWebアプリケーションです。

## 🎯 目的

AIChatサービスの主な目的は以下の通りです：

1. **使いやすいAIインターフェースの提供**: 直感的で使いやすいチャットUIを通じて、AIの能力にアクセスできるようにします
2. **一貫性のあるユーザー体験**: 他のTachyon Appsサービスと統合された一貫性のあるデザインと操作性を提供します
3. **パフォーマンスと拡張性**: 高速なレスポンスと将来の機能拡張を見据えた設計を実現します
4. **カスタマイズ性**: 各ユーザーのニーズに合わせた機能のカスタマイズを可能にします

## 🏗️ アーキテクチャ

AIChatは次のようなアーキテクチャで構成されています：

```
クライアント ←→ API層 ←→ LLMサービス
    ↑             ↓  
    └─────→ データストア
```

### コンポーネント構成

1. **フロントエンド**:
   - Next.js (App Router)
   - React コンポーネント
   - Vercel AI SDK
   - shadcn/ui コンポーネント

2. **バックエンド**:
   - Next.js API Routes
   - LLMプロバイダー連携層
   - データストア連携層

3. **データストア**:
   - チャットルーム管理
   - メッセージ履歴
   - ユーザー設定

## 🚀 主要機能

AIChatサービスは以下の主要機能を提供します：

### チャット機能

- リアルタイムAI対話（ストリーミングレスポンス）
- マークダウン対応のリッチテキスト表示
- コードブロックのシンタックスハイライト
- メッセージの編集と削除

### チャットルーム管理

- 複数のチャットルーム作成と管理
- チャット履歴の保存と閲覧
- チャットルームの名前変更と削除

### カスタマイズ機能

- モデル選択（複数のAIモデルに対応）
- システムプロンプトのカスタマイズ
- テーマ設定（ライト/ダークモード）

### エージェント機能

- 複雑なタスクを自動実行するAIエージェント
- ツール実行とその結果の視覚化
- 対話型のタスク承認とフィードバック
- リアルタイムの進捗状況表示

## 🛠️ 使用技術

| 分野 | 技術 |
|-----|------|
| フレームワーク | Next.js, React |
| スタイリング | Tailwind CSS, shadcn/ui |
| AI統合 | Vercel AI SDK |
| データ取得 | SWR |
| テスト | Vitest, Playwright, Storybook |
| ビルドツール | Turborepo |

## 🔄 開発ワークフロー

AIChatの開発は以下のワークフローに従います：

1. **機能設計**: 要件定義、ワイヤーフレーム作成
2. **コンポーネント開発**: UIコンポーネントの実装
3. **API実装**: バックエンドAPIエンドポイントの開発
4. **統合**: フロントエンドとバックエンドの統合
5. **テスト**: 単体テスト、インタラクションテスト、E2Eテスト
6. **デプロイ**: CI/CDパイプラインを通じたデプロイ

AIChatサービスでは、今後以下の機能強化を予定しています：
