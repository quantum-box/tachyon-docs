# Library ガントチャートビュー

## 概要

Library アプリの Data ビューにガントチャートビューを追加し、日付プロパティタイプ（Date）を利用してプロジェクト管理やタスク管理を視覚的に行えるようにしました。

## 機能

### ガントチャートビュー

- **ビューモード**: Data ビューに `gantt` モードを追加
- **開始日・終了日プロパティ選択**: ガントチャート表示に使用する開始日・終了日プロパティを選択可能
- **タイムライン表示**: 月/週/日ヘッダーとグリッド線による視覚的なタイムライン表示
- **タスクバーのドラッグ＆ドロップ**: タスクバーをドラッグして日付を変更、リサイズハンドルで期間を変更
- **ズーム機能**: 日/週/月/年の4段階表示切り替え
- **日付編集ダイアログ**: タスクバークリックで DatePicker による日付編集

### 日付プロパティタイプ

- **Date プロパティタイプ**: 日付のみ（時刻なし）を保存するプロパティタイプ
- **ISO 8601 フォーマット**: `YYYY-MM-DD` 形式で保存・表示
- **DatePicker コンポーネント**: shadcn/ui 準拠の DatePicker で直感的な日付選択

## 技術仕様

### バックエンド

- **PropertyType**: `Date` を追加
- **PropertyDataValue**: `Date(chrono::NaiveDate)` を追加
- **GraphQL API**: `DateValue` 型を追加
- **日付バリデーション**: ISO 8601 形式のパース・バリデーション

### フロントエンド

- **ガントチャート実装**: カスタム実装（`date-fns` を使用）
- **DatePicker**: shadcn/ui の DatePicker コンポーネント
- **日付ライブラリ**: `date-fns` を使用したタイムライン計算

### データ構造

```typescript
interface GanttTask {
  id: string
  name: string
  startDate: Date
  endDate: Date
  progress?: number // 0-100 (将来の拡張)
  dependencies?: string[] // 依存タスクID (将来の拡張)
}
```

## 使い方

1. **日付プロパティの作成**: プロパティ作成時に `Date` タイプを選択
2. **データに日付を設定**: データ詳細ページで DatePicker を使用して日付を設定
3. **ガントチャートビューに切り替え**: Data ビューのビューモードから `gantt` を選択
4. **開始日・終了日プロパティを選択**: ガントチャート表示に使用するプロパティを選択
5. **タスクの操作**:
   - タスクバーをドラッグして日付を変更
   - タスクバーの端をドラッグして期間を変更
   - タスクバーをクリックして日付編集ダイアログを開く
   - Zoom In/Zoom Out ボタンで表示範囲を調整

## 将来の拡張機能

- **依存関係の表示**: タスク間の依存関係を矢印で表示
- **進捗率表示**: タスクの進捗率（0-100%）を表示
- **パフォーマンス最適化**: 大量データでの仮想スクロール対応
